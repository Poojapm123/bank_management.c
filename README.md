
# 🏦 Bank Management System (C)

This is a simple Bank Management System developed using C programming.  
It allows users to manage bank accounts with basic operations.

## 🚀 Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance

## 🛠️ Technologies Used
- C Programming
- File Handling

## 📂 Project Structure
- bank_management.c
- bank.txt (auto-created for storing data)

## ▶️ How to Run
1. Compile the code using a C compiler
2. Run the program
3. Choose options from the menu
